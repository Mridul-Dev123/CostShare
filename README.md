# CostShare Studio

CostShare Studio is a production-focused React app for proportional expense settlement.

## Features

- Add participants with numeric values or arithmetic expressions.
- Calculate proportional final settlement from a target total.
- Edit/delete entries with local persistence.
- Export results table to PDF.

## Development

Install dependencies:

```bash
npm install
```

Start local dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Run tests:

```bash
npm run test:run
```
